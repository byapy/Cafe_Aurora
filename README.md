# Café Aurora ☕✨

Bienvenido al repositorio de **Café Aurora**, un sitio web moderno, elegante y completamente responsivo diseñado para ofrecer la mejor experiencia visual a los amantes del café.

## 🌟 Descripción
Este proyecto es una plataforma web integral para una cafetería de alta calidad. El sitio combina un diseño estético con funcionalidad práctica, permitiendo a los usuarios explorar la historia de la marca, consultar un menú detallado, ver promociones vigentes y ponerse en contacto con la tienda.

## 🚀 Características Principales
- **Diseño Responsivo:** Adaptado con *Media Queries* específicas para dispositivos móviles (580px), tablets (870px) y pantallas grandes (980px+).
- **Menú Interactivo:** Implementación de acordeones dinámicos de **Bootstrap 5** para organizar bebidas, comida y postres de forma compacta.
- **Estética Premium:** Uso de efectos visuales modernos como el desenfoque (*blur*) en fondos, bordes redondeados y una paleta de colores basada en tonos tierra y rosados (#CB7885, #893941).
- **Navegación Dinámica:** Menú de navegación con efectos *hover* y transiciones suaves.

## 🛠️ Tecnologías Utilizadas
- **HTML5:** Marcado semántico para SEO y accesibilidad.
- **CSS3 Avanzado:** Uso de **CSS Grid** y **Flexbox** para layouts complejos y responsivos.
- **Bootstrap 5:** Integración vía CDN para componentes interactivos y rejilla de contacto.
- **JavaScript / jQuery:** Soporte para la interactividad de los componentes de Bootstrap.

## 📂 Estructura del Proyecto
- `index.html`: Home con banner principal y sección de logros (Grid).
- `nosotros.html`: Sección "Quiénes Somos" con diseño de tarjetas de equipo.
- `menu.html`: El corazón del sitio, con un desglose visual de productos.
- `galeria.html`: Galería fotográfica de productos y local.
- `promociones.html`: Espacio para ofertas especiales.
- `contacto.html`: Formulario validado con estilos de Bootstrap y enlaces a redes sociales.
- `/styles`: 
    - `inicio.css`: Estilos globales y layouts principales.
    - `menu-promociones.css`: Estilos específicos para la visualización de productos.
- `/imagenes` & `/imagen`: Recursos gráficos del sitio.

## 📖 Cómo visualizar el proyecto
1. Clona este repositorio o descarga los archivos.
2. Asegúrate de tener conexión a internet (necesaria para cargar Bootstrap y FontAwesome vía CDN).
3. Abre el archivo `index.html` en tu navegador.
